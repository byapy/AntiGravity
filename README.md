# AntiGravity prueba Netflix Clone

![Intecap-Logo](https://intecap.edu.gt/wp-content/uploads/2021/06/logo-intecap-760x336.png)
### Descripción
Esta es una pagina 100% generada por ia mediante Gemini en el IDE Antigravity de Google.
Como practica en el manejo de herramientas de inteligencia artificial.
#### Tecnologías Utilizadas:
| HTML | CSS |  JavaScript  | Bootstrap
|--|--|--|--|
| Para formar la estructura | CSS externo para personalización | Scripts para funcionalidad general de formularios y diseños dinamicos | Framework en CDN utilizado para diseños rápidos y genéricos 


#### Características 
- Hero banner en carrusel que cambia automáticamente
- Secciones de películas y series en carrusel horizontal. Nombre y titulo del contenido audiovisual.
- Sección de LIVE con boton de reproducir ahora, hora de inicio, ubicación y titulo.
- Sección de próximos eventos en vivo
- 3 subpáginas dedicadas a categorías como: Series, Películas y Novedades Populares 
- Boton para pagina administrador de perfiles 
- Pagina de pagos con un formulario con validaciones, distintos métodos de pago incluidos.
